# NetScope – Local Network & Port Scanner

NetScope is a Python-based tool that helps users monitor their local system and network.

It can:
- Detect open ports on the local machine
- Scan the local network (LAN)
- Identify devices connected to the WiFi network
- Display IP and MAC addresses of connected devices

This project is useful for learning networking, cybersecurity basics, and Python network programming.

---

## 🚀 Features

- Detect open ports running on your computer
- Scan devices connected to the same WiFi network
- Display IP and MAC address of each device
- Lightweight and easy to run
- Built using Python

##  Technologies Used

- Python

- scapy
